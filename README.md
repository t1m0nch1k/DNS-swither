# 🦊 RedPandaDNS

**RedPandaDNS** — это интуитивно понятное, быстрое и современное Android-приложение для динамического переключения DNS-серверов в один клик. Приложение объединяет передовые сетевые технологии с приятным визуальным дизайном, украшенным милым маскотом — Красной Пандой.

[![Скачать APK](https://img.shields.io/badge/Скачать-APK%20v1.0.0-blue?style=for-the-badge&logo=android&logoColor=white)](https://github.com/t1m0nch1k/DNS-swither/releases/tag/1.0.0)

Приложение спроектировано с использованием **Kotlin**, **Jetpack Compose** и современных архитектурных рекомендаций Google (MVVM, Clean Architecture).

## 🚀 Основные возможности

### 1. Два мощных движка переключения DNS
* **Локальный VPN движок (`Local VPN Engine`)**: Сверхлегкий фоновый сервис `VpnService`, который перехватывает исключительно DNS-запросы вашего устройства и перенаправляет их на выбранный DNS-сервер.
  * *Плюсы*: Работает «из коробки», не требует рут-прав, не требует ADB-настройки, совместим со всеми версиями Android.
* **Системный Private DNS движок (`Private DNS Mode`)**: Взаимодействует напрямую с системным механизмом «Частного DNS» (Private DNS / DNS-over-TLS), встроенным в Android 9+.
  * *Плюсы*: Нулевое потребление батареи, максимальное быстродействие на уровне ядра ОС.
  * *Требование*: Требуется предоставить разрешение через ADB или права Device Owner.

### 2. Управление серверами
* **Популярные DNS по умолчанию**: В приложение сразу предустановлены готовые профили:
  * **Cloudflare** (`1.1.1.1` | `1dot1dot1dot1.cloudflare-dns.com`)
  * **Google Public DNS** (`8.8.8.8` | `dns.google`)
  * **AdGuard DNS** (для блокировки рекламы и трекеров | `dns.adguard-dns.com`)
  * **Quad9** (безопасное шифрование | `dns.quad9.net`)
  * **CleanBrowsing** (для семейного контроля | `security-filter-dns.cleanbrowsing.org`)
* **Пользовательские DNS**: Свободное добавление личных DNS серверов с указанием названия, DoT хоста и IP-адресов. Данные надежно сохраняются в локальной базе данных **Room SQlite**.

### 3. Интерактивный Виджет (App Widget)
* Удобный виджет для домашнего экрана позволяет контролировать подключение и переключаться между избранными конфигурациями в одно касание, не открывая главное приложение.

## 🛠 Настройка системного Private DNS движка

Для полноценной работы системного движка **Private DNS** приложению необходимы особые привилегии на изменение скрытых настроек системы (`WRITE_SECURE_SETTINGS`) или права администратора устройства (`Device Owner`). 

Установите утилиту `adb` на компьютер, подключите телефон по USB с включенной «Отладкой по USB» и выполните одну из команд ниже:

### Вариант А: Предоставление Secure Settings (Рекомендуемый)
Запустите консоль и введите команду:
```bash
adb shell pm grant com.aistudio.privatedns.swrxtw android.permission.WRITE_SECURE_SETTINGS
```
*После этого перезапустите приложение. Вы сможете переключать системный Private DNS прямо из интерфейса.*

### Вариант Б: Режим владельца устройства (Device Owner)
Если вам нужна максимальная глубина системной интеграции на корпоративных или личных устройствах, вы можете настроить права владельца:
```bash
adb shell dpm set-device-owner com.example/.MyDeviceAdminReceiver
```

## 💻 Сборка и запуск проекта

Проект собирается стандартными средствами Gradle (Kotlin DSL).

### Требования
* JDK 11 или выше
* Android SDK 28+ (Min SDK 28, Target SDK 36, Android 16 Api 36 Ready)
* Android Studio (Koala / Ladybug или новее)

### Сборка через консоль
```bash
# Клонировать репозиторий
git clone https://github.com/your-username/RedPandaDNS.git
cd RedPandaDNS

# Сборка дебаг-версии (APK)
gradle assembleDebug
```

Готовый APK файл будет доступен в директории:
`app/build/outputs/apk/debug/app-debug.apk`

## 📂 Структура исходного кода

Благодаря архитектуре MVVM исходный код легко читать и масштабировать:
* `DnsVpnService.kt` — Легковесный системный VPN-сервис для интеграции локального DNS-проксирования.
* `DnsController.kt` — Системный контроллер, отвечающий за логику переключения DNS в режимах VPN и Private DNS.
* `MainActivity.kt` — Главный экран приложения, реализованный на Jetpack Compose с декларативной разметкой.
* `DnsViewModel.kt` — Прослойка управления реактивными состояниями и бизнес-логикой.
* `AppDatabase.kt` / `DnsDao.kt` — Локальное хранилище серверов на базе Room.
* `DnsWidgetProvider.kt` — Реализация нативного интерактивного виджета для статус-бара рабочего стола.

## 🔒 Безопасность и конфиденциальность

1. **Локальная обработка данных**: Все DNS-запросы в режиме VPN обрабатываются локально на вашем смартфоне. Приложение **не собирает**, **не сохраняет** и **не передает** ваши личные данные, аналитику или историю сетевых соединений.
2. **Открытый исходный код**: Вы полностью контролируете, какие процессы происходят во время работы программы.
3. **Безопасное шифрование**: DoT-хосты позволяют защитить ваши DNS-запросы от перехвата и подмены со стороны провайдеров связи.
